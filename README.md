# Claude AI for iOS

🤖 专为 iOS Safari 和 Chrome 浏览器优化的 Claude AI 对话界面

## ✨ 特性

- 🍎 完美支持 iOS Safari 和 Chrome 浏览器
- 💬 简洁优雅的对话界面
- 🎨 响应式设计，适配各种屏幕尺寸
- 💾 自动保存对话历史
- 🌙 支持暗色模式
- ⚡ 基于 Claude 4.5 最新模型

## 🚀 在线使用

访问部署地址即可直接使用（部署后会生成）

## 📱 使用方法

1. 在 iOS 设备上打开部署后的网址
2. 在输入框输入你的问题
3. 点击发送按钮或按回车键
4. AI 会自动回复你的问题

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- Claude 4.5 API
- LocalStorage 用于对话历史保存

## 📦 部署到 Vercel

### 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/chinastreamagency-alt/claude-for-apple-ios)

### 手动部署

1. Fork 这个仓库
2. 登录 [Vercel](https://vercel.com)
3. 点击 "Import Project"
4. 选择你 fork 的仓库
5. 点击 "Deploy"

## 🎯 功能说明

- **对话功能**：与 Claude AI 进行自然语言对话
- **清空对话**：点击右上角"清空对话"按钮可清除历史记录
- **对话保存**：对话会自动保存在浏览器本地，刷新页面不会丢失
- **移动优化**：专门为触屏设备优化的交互体验
